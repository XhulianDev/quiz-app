# Quiz App

An interactive quiz application designed to test knowledge across various subjects.

## Description

This application features questions from a diverse range of fields:
* **Natural Sciences:** Physics, Biology, Geography
* **Social Sciences:** History, Psychology, Anthropology
* **Humanities:** Literature, Philosophy, Visual Arts

It is optimized for all devices, ensuring a **responsive user experience**.

## Recent Refactor & Improvements

* **Code Optimization:** Removed redundant CSS and consolidated media queries for a cleaner codebase.
* **Modern Layout:** Implemented Flexbox to handle element alignment and responsiveness efficiently.
* **Modular Architecture:** Split the project into logical files (Separation of Concerns) for better maintainability.
* **Dynamic Content:** Questions are now handled via JavaScript modules, making the app easier to scale.

## Technologies Used

* **HTML5**: For structuring the core content.
* **CSS3**: For styling and layout.
* **JavaScript**: For all interactive functionalities and quiz logic.
